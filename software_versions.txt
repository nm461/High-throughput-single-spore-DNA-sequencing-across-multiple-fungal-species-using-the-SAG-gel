## Containerized tools to verify from the final run environment

The following tools are run from Apptainer/Singularity images in SIF_DIR. Record
the exact executable version from the image used for the archived analysis:

fastp.sif
bwa.sif
samtools.sif
rasusa.sif
kraken2.sif
pilon.sif
seqkit.sif
busco_v6.0.0_cv1.sif
mosdepth.sif
sourmash.sif

Suggested commands:

```bash
singularity exec "$SIF_DIR/fastp.sif" fastp --version
singularity exec "$SIF_DIR/bwa.sif" bwa 2>&1 | head -5
singularity exec "$SIF_DIR/samtools.sif" samtools --version
singularity exec "$SIF_DIR/rasusa.sif" rasusa --version
singularity exec "$SIF_DIR/kraken2.sif" kraken2 --version
singularity exec "$SIF_DIR/pilon.sif" java -jar /pilon/pilon.jar --version
singularity exec "$SIF_DIR/seqkit.sif" seqkit version
singularity exec "$SIF_DIR/busco_v6.0.0_cv1.sif" busco --version
singularity exec "$SIF_DIR/mosdepth.sif" mosdepth --version
singularity exec "$SIF_DIR/sourmash.sif" sourmash --version
java -jar "$PICARD_JAR" MarkDuplicates --version
```

## Python packages installed by the scripts

The scripts create a local virtual environment and install the following
packages when needed:

pandas
numpy
matplotlib
seaborn
biopython
scipy
PyPDF2

For a final archive, replacing the unpinned package install lines with a
frozen `requirements.txt` from the analysis environment is recommended:

```bash
source "$OUT_ROOT/shared_venv/bin/activate"
pip freeze > requirements.txt
```
